Full-Stack Travel Booking Platform
ReactJS · Node.js · Google Maps API · MySQL · JWT
Authentication

1.A comprehensive, end-to-end travel booking platform that allows users to book cabs,
hotels, and travel packages — all within a single unified interface. Think of it as a mini
MakeMyTrip built from scratch.
2.Frontend 
— Built entirely in ReactJS with a modular component architecture, ensuring
reusable UI elements and clean separation of concerns across booking workflows
3.Backend 
— Node.js server with a decoupled, modular design — each service (cab, hotel,
packages) operates independently, meaning one service can be updated or deployed
without breaking the others
4.Real-Time Routing 
— Integrated Google Maps API and Google Places API for live cab
routing, real-time ETA tracking, and location-based search — users see their driver's
movement in real time
5.Authentication 
— JWT (JSON Web Token) based user authentication system — secure
login, session management, and protected dashboard routes
6.Database 
— MySQL relational database handling user profiles, booking records,
transaction history, and route data
